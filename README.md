# DisjointSet
to run the program execute 
```
$ python3 disjoint_set.py input_file_path
```
the above command will produce a file output.txt which content is the returns of the program execution

<br>

If you want to get the answer for multiple input files use
```
$ python3 disjoint_set.py input_file_1 input_file_2 .... input_file_n
```

<br>

you can also specifiy the output file by providing the file path (use the flags -o or --output)
```
$ python3 disjoint_set.py input_file_1 input_file_2 -o out.txt
```
