# DisjointSet
to run the program execute 
```
$ python3 disjoint_set.py input_file_path
```
the above command will produce a file output.txt whit the reeturns of the program exucuion

<br>

If you want to get the anwser for multiple input files use
```
$ python3 disjoint_set.py input_file_1 input_file_2 .... input_file_n
```

<br>

you can also specifiy the output file by promiding the file path (use the flags -o or --output)
```
$ python3 disjoint_set.py input_file_1 input_file_2 -o out.txt
```
